📚 Quick Guide

Role Permissions:

  • Admin: Full access - Can create users, manage all data, send emails
  • DEO: Data management - Can add/edit students, upload files, send emails
  • HOD: Read-only - Can view reports, dashboard, and analytics

How to Create New User:

  1. Click "+ New User" button above
  2. Fill in username, password, role, and department
  3. Click "Register User"
  4. Save the QR code for 2FA setup
  5. Share credentials with the new user

Default Credentials:

  • Admin: admin / admin123 / OTP: 000000
  • DEO: deo_cse / cse123 / OTP: 000000
  • HOD: hod_cse / hod123 / OTP: 000000
